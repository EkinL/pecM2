# UML (PlantUML)

Diagrammes generes a partir du code actuel du projet.

## Fichiers

- `01_system_context.puml` : vue globale (actors + systemes)
- `02_next_components.puml` : architecture interne Next.js
- `03_mobile_components.puml` : architecture interne iOS SwiftUI
- `04_sequence_message_flow.puml` : sequence envoi message + reponse IA
- `05_sequence_rgpd_flow.puml` : sequence demande suppression compte (RGPD)

## Rendu rapide

Depuis la racine du repo:

```bash
plantuml docs/uml/*.puml
```

Ou dans VS Code avec l extension PlantUML.
